# 엑셀 다운로드 파일 서식 개선 - 요구사항

## 배경
현재 엑셀 다운로드 시 숫자 및 비율 값이 **문자열(텍스트)**로 저장되고 있어,
엑셀에서 정렬/필터/계산 등 후속 작업이 불가능하고 서식이 일관되지 않음.

## 요구사항

### REQ-1: 금액/수량 숫자 서식
- 모든 숫자(금액, 수량 등)는 **숫자 타입**으로 저장한다.
- 엑셀 셀 서식으로 **3자리 쉼표 구분자**(천 단위 구분)를 적용한다.
- 예시: `100000` → 셀에 `100,000`으로 표시 (실제 값은 숫자 100000)

### REQ-2: 백분율 서식
- `%`가 포함된 항목(달성률, 수량비중, 금액비중, 환불율 등)은 **숫자 타입**으로 저장한다.
- 엑셀 셀 서식으로 **백분율 소수점 첫째 자리**까지 표시한다 (소수점 둘째 자리에서 반올림).
- 원본 데이터가 소수(0~1 범위)인 경우: 그대로 저장 후 엑셀 `%` 서식 적용
- 원본 데이터가 이미 백분율(0~100 범위)인 경우: 100으로 나눈 후 저장하여 엑셀 `%` 서식 적용
- 예시: `0.364235725` → 셀에 `36.4%`로 표시

### REQ-3: 영향 범위
아래 시트의 해당 컬럼에 적용:

| 시트명 | 컬럼 | 서식 |
|--------|------|------|
| KPI 수치 | 최종결제액, 결제금액, 결제수, 쿠폰합계, 환불액 | 숫자 쉼표 |
| KPI 수치 | 달성률, 환불율 | 백분율 |
| 일별 판매성과 | 결제금액, 환불금액, 최종결제액, 쿠폰합계 | 숫자 쉼표 |
| 상품별 성과 | 판매수량, 결제금액 | 숫자 쉼표 |
| 상품별 성과 | 수량비중, 금액비중, 환불율 | 백분율 |
| 행사 컨텍스트 | 매출목표 | 숫자 쉼표 |
